To run our code:
1. type java BigRun in the terminal
- This will calculate the average hit rates for each cache replacement policy for cache
sizes 10 to 200 with sequences a the uniform distribution and then a Zipf distribution.
-The program will save the average hit rates into 8 csv files based on
the cache replacement policy and the distribution the sequences were drawn from.
2. type java GardnerTest
- This will calculate the average hit rates for each cache replacement policy for cache
sizes 10 to 200 with sequences from a dependent distribution.
-The program will save the average hit rates into 4 csv files based on
the cache replacement policy.
